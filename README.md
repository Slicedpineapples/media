# Share

Share is a lightweight media hosting and link-sharing platform by Software Pulses.

It provides a centralized workspace for storing approved media assets, then converting those assets into clean shareable URLs.

## Overview

Share is designed for teams and individuals who need a straightforward asset management layer for organizing media and distributing files through direct links.

The workflow is intentionally minimal: upload an asset, generate a URL, copy it, and share it across devices, teams, or platforms.

Supported uploads are intentionally limited to media formats such as images, audio, and video. General documents and arbitrary file attachments are not part of the product scope.

## Highlights

- Centralized storage for approved media assets
- Direct shareable URLs for every uploaded file
- Folder-based organization with previews and quick actions
- Responsive interface across desktop and mobile
- Progressive web app support for a more native experience

## Security

Share applies upload security screening before files are accepted. Uploads are checked against an allowlist of supported media formats, validated against their declared MIME type, and inspected using file-signature checks to help block unsupported, disguised, or potentially harmful files.

This reduces malware risk by preventing arbitrary file uploads and limiting storage to known media types.

## Product Goal

Share keeps media distribution direct, reliable, and easy to understand: upload once, generate a link, and share without friction.
